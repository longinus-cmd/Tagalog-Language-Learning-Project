Tagalog Language Learnig Game, repository containing all concurrent and worked on programs.
This code was worked made through the usage of Claude AI.
The work done here does not cost money and will never be, and is created entirely for fun.

Current Programs:
Tagalog Jeopardy, A jeopardy-style game where the user must use tagalog vocabulary in order to gain points.
Salita Shooter, a space invaders-esque game where the user must shoot the corect word before it drops down to the bottom of the screen.
